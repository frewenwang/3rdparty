Enum Documentation¶

enum SnpeUdo_CoreType_t¶

An enum which holds the UDO library Core type . Designed to be used as single values or combined into a bitfield parameter (0x1, 0x2, 0x4, etc)

Values:

SNPE_UDO_CORETYPE_UNDEFINED = 0x00¶

Library target IP Core is undefined.

UDO_CORETYPE_UNDEFINED = 0x00¶
SNPE_UDO_CORETYPE_CPU = 0x01¶

Library target IP Core is CPU.

UDO_CORETYPE_CPU = 0x01¶
SNPE_UDO_CORETYPE_GPU = 0x02¶

Library target IP Core is GPU.

UDO_CORETYPE_GPU = 0x02¶
SNPE_UDO_CORETYPE_DSP = 0x04¶

Library target IP Core is DSP.

UDO_CORETYPE_DSP = 0x04¶
SNPE_UDO_CORETYPE_LAST = 0xFFFFFFFF¶
UDO_CORETYPE_LAST = 0xFFFFFFFF¶
