Enum Documentation¶

enum QnnGraph_Error_t¶

QNN Graph API result / error codes.

Values:

QNN_GRAPH_MIN_ERROR = 6000¶
QNN_GRAPH_NO_ERROR = 0¶

Qnn Graph success.

QNN_GRAPH_ERROR_UNSUPPORTED_FEATURE = QNN_COMMON_ERROR_NOT_SUPPORTED¶

There is optional API component that is not supported yet. See QnnProperty.

QNN_GRAPH_ERROR_MEM_ALLOC = QNN_COMMON_ERROR_MEM_ALLOC¶

General error relating to memory allocation in processing graph API.

QNN_GRAPH_ERROR_INVALID_ARGUMENT = 6000 + 0¶

An argument to QNN API is deemed invalid by a backend.

QNN_GRAPH_ERROR_INVALID_HANDLE = 6000 + 1¶

Invalid graph handle.

QNN_GRAPH_ERROR_GRAPH_DOES_NOT_EXIST = 6000 + 2¶

No graph with specified info is registered in the backend.

QNN_GRAPH_ERROR_INVALID_NAME = 6000 + 3¶

Invalid or duplicate graph name.

QNN_GRAPH_ERROR_INVALID_TENSOR = 6000 + 4¶

Invalid or NULL QNN tensor.

QNN_GRAPH_ERROR_INVALID_OP_CONFIG = 6000 + 5¶

Some elements in the op config data are invalid.

QNN_GRAPH_ERROR_SET_PROFILE = 6000 + 6¶

Failure to set profile.

QNN_GRAPH_ERROR_UNCONNECTED_NODE = 6000 + 7¶

Node added before its dependent node(s)

QNN_GRAPH_ERROR_CREATE_FAILED = 6000 + 20¶

Failure in creating graph with specified configuration.

QNN_GRAPH_ERROR_OPTIMIZATION_FAILED = 6000 + 21¶

Graph couldn’t be optimized with specified list of ops or config.

QNN_GRAPH_ERROR_FINALIZE_FAILED = 6000 + 22¶

Graph finalize failed.

QNN_GRAPH_ERROR_GRAPH_NOT_FINALIZED = 6000 + 23¶

Graph attempted to be executed before being finalized.

QNN_GRAPH_ERROR_GRAPH_FINALIZED = 6000 + 24¶

Graph attempted to be modified after being finalized.

QNN_GRAPH_ERROR_EXECUTION_ASYNC_FIFO_FULL = 6000 + 25¶

FIFO queue cannot register any more async execution requests.

QNN_GRAPH_ERROR_SIGNAL_OBSERVED = 6000 + 30¶

A control signal object was provided to a call, but that signal object is already observed by another call.

QNN_GRAPH_ERROR_ABORTED = 6000 + 31¶

Return when a call is aborted early due to a QnnSignal_abort call issued to the observed signal object.

QNN_GRAPH_MAX_ERROR = 6999¶
QNN_GRAPH_ERROR_UNDEFINED = 0x7FFFFFFF¶
