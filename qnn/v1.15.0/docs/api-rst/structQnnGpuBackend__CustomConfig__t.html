Struct Documentation¶

struct QnnGpuBackend_CustomConfig_t

A struct which defines the QNN GPU backend custom configuration options. Objects of this type are to be referenced through QnnBackend_CustomConfig_t.

Note

DEPRECATED: Use QnnGpuContext_CustomConfig_t instead.

Public Members

const char *kernelRepoDir

The kernel disk cache directory. Must be non-null and is copied into the backend.

uint8_t invalidateKernelRepo

If non-zero, the kernel disk cache will be ignored when initializing.
