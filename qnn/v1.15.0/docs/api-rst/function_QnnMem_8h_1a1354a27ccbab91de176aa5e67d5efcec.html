Function QnnMem_deRegister¶

  • Defined in File QnnMem.h

Function Documentation¶

Qnn_ErrorHandle_t QnnMem_deRegister(const Qnn_MemHandle_t *memHandles, uint32_t numHandles)

Deregister a memory which was registered via QnnMem_register and invalidates memHandle for the given backend.

Parameters

  • [in] memHandles: Array of memory handles to be deregistered

  • [in] numHandles: Number of memory handles in the array

Note

memHandles parameter: Upon failure, all valid handles within memHandles will still be deregistered.

Return

Error code:

  • QNN_SUCCESS: memory was successfully deregistered

  • QNN_COMMON_ERROR_BACKEND_NOT_INITIALIZED: backend not initialized

  • QNN_MEM_ERROR_NOT_SUPPORTED: backend does not support this API

  • QNN_MEM_ERROR_INVALID_ARGUMENT: memHandles is NULL

  • QNN_MEM_ERROR_INVALID_HANDLE: a handle within memHandles is NULL/invalid
