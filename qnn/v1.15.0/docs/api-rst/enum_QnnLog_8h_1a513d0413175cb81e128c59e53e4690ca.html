enum QnnLog_Error_t¶

QNN Log API result / error codes.

Values:

QNN_LOG_MIN_ERROR = 11000¶
QNN_LOG_NO_ERROR = 0¶

Qnn Log success.

QNN_LOG_ERROR_MEM_ALLOC = QNN_COMMON_ERROR_MEM_ALLOC¶

General error relating to memory allocation in Log API.

QNN_LOG_ERROR_ALREADY_INITIALIZED = 11000 + 0¶

Function could not be called because logging has already been successfully initialized.

QNN_LOG_ERROR_NOT_INITIALIZED = 11000 + 1¶

Function could not be called because logging has not yet been initialized.

QNN_LOG_ERROR_INITIALIZATION = 11000 + 2¶

Unable to initialize logging.

QNN_LOG_ERROR_INVALID_ARGUMENT = 11000 + 3¶

Invalid argument passed.

QNN_LOG_MAX_ERROR = 11999¶
QNN_LOG_ERROR_UNDEFINED = 0x7FFFFFFF¶
