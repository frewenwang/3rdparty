enum QnnSaver_ConfigOption_t¶

This enum contains the supported config options for Saver.

Values:

QNN_SAVER_CONFIG_OPTION_OUTPUT_DIRECTORY = 0¶

Configuration of the location Saver outputs.

QNN_SAVER_CONFIG_OPTION_APPEND_TIMESTAMP = 1¶

Configuration of timestamp appended to Saver outputs.

QNN_SAVER_CONFIG_OPTION_BACKEND_ID = 2¶

Configuration indicating to Saver which backend to interpret custom configs as. This option should only be used if you are providing custom configs to QNN APIs that support them (e.g. QnnBackend_Initialize()) and you want these custom configs to be recorded by Saver.

QNN_SAVER_CONFIG_OPTION_UNDEFINED = 0x7FFFFFFF¶
