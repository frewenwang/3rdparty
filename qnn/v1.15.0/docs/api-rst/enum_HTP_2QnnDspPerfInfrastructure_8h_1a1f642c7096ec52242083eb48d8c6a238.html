Enum Documentation¶

enum QnnDspPerfInfrastructure_VtcmUsageFactor_t

This enum defines all the possible vtcm usage configuration. These settings apply only for on-target libraries.

Values:

QNN_DSP_PERF_INFRASTRUCTURE_VTCM_USE_FULL = 1

use all the vtcm available on target

QNN_DSP_PERF_INFRASTRUCTURE_VTCM_USE_MIN = 2

use bare minimal vtcm available on target. This is not supported in the current release.

QNN_DSP_PERF_INFRASTRUCTURE_VTCM_USE_UNKNOWN = 0x7fffffff
