Struct Documentation¶

struct QnnProperty_OperationName_t

Struct which encapsulates the fully-qualified name of an operation. packageName is the op package to which the operation belongs, name is the type name of the operation, and target is the intended target platform for the combination of domain and operation name. Target may be unused (NULL) by some backends.

Public Members

const char *packageName
const char *name
const char *target
