Function Documentation¶

ModelError_t qnn_wrapper_api::getGraphInfoFromModels(QnnModel *models, uint32_t numModels, GraphInfoPtr_t **graphsInfo)

A helper function to convert QnnModel objects to Graph struct for qnn_model c interface.

Parameters

  • [in] models: List of QnnModel objects

  • [in] numModels: The number of elements in above models object

  • [out] graphsInfo: The corresponding array of Graph object for each of the above model objects(note: this function will malloc memory needed to store the struct objects. Following free shall be invoked when objects are no longer needed.

Return

Error code
