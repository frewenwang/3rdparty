ModelError_t qnn_wrapper_api::deepCopyQnnTensorWrappers(Qnn_TensorWrapper_t source, Qnn_TensorWrapper_t &destination)

Deep Copies QnnTensorWrapper structs to a pointer array destination location. Note: The copy will be stored on the heap and as such requires caller to make appropriate free call(s) using function below. Note 2: deepCopy is only done for metadata.

Parameters

  • [in] source: tensor object to copy from

  • [in] destination: tensor object to copy to
