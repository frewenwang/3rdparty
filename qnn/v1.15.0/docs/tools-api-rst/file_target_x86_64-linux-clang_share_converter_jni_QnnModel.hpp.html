ModelError_t initialize(const Qnn_ContextHandle_t &context, const char *graphName, bool debug, uint8_t doNodeValidations = 1, bool assignTensorIds = false, const QnnGraph_Config_t **graphConfigs = NULL)

Creates a Qnn Graph within given context.

Parameters

  • [in] context: A handler to the context where the model’s graph would be created.

  • [in] graphName: The name to use for creating a graph in the context provided.

  • [in] debug: If flag is true, sets all tensors created in model to be QNN_TENSOR_TYPE_APP_READ, essentially overwriting what is set in Qnn_TensorType.

  • [in] doNodeValidations: If flag is set, all nodes added with addNode call will be validated by Backend

  • [in] assignTensorIds: If flag is set, passed in tensorWrappers will be assigned a unique id prior to creating tensor in backend using addTensor(). If Qnn_Tensor {.id} field is already set, it will be overwritten.

  • [in] graphConfigs: Array of graph configurations to use for creating the QNN Graph. Default: NULL

ModelError_t addNode(char *name, char *packageName, char *type, Qnn_ParamWrapper_t *params, uint32_t numOfParams, char **inputNames, uint32_t numOfInputs, Qnn_TensorWrapper_t *outputWrappers, uint32_t numOfOutputs)

A wrapper function to create a node in class’s graph.

Parameters

  • [in] name: The node name to use (e.g. my_graph_conv_1)

  • [in] packageName: The node package name (e.g. qti.aisw)

  • [in] type: The QNN_OP_QNN_OP_H node type (e.g. QNN_OP_ARGMAX)

  • [in] params: A struct object containing all the params for the node to be added. For tensorparam case. The tensor will be created within the function and the data will be retreived from the binary blob to set the tensor data.

  • [in] numOfParams: The number of elements in above params object

  • [in] inputNames: List of tensor names for inputs to node. Note: the corresponding qnn tensor objects must be created within this instance prior to being listed as input to a node

  • [in] numOfInputs: The number of elements in above inputNames object

  • [in] outputWrappers: List of Qnn_TensorWrapper_t objects for outputs from node. Note1: the corresponding qnn tensor objects will be created in function and must not already exist. Note2: the output names must be unique per graph

  • [in] numOfOutputs: The number of elements in above outputs object

Return

Error code
