QNN GPU API Revision History¶

QNN GPU API Version

Introduced in QNN SDK Version

Type of Change

Description

2.5.0

1.15.0

API Updates

  • QnnGpuContext.h: Move GPU kernel repo to QNN Context

2.4.0

1.14.0

API Updates

  • QnnGpuOpPackage.h: Support tensor type constraints in super node

OpDef Updates

  • Relaxing BatchNorm Validator rank condition to support rank in [1,4].

2.3.0

1.13.0

API Updates

  • QnnGpuContext.h: Support GL buffers.

OpDef Updates

  • Support Concat operation with rank = 1 and input dimensions > 16384.

  • Support axis = 0 for rank = 4 input in Split operation.

  • Added support for Nv12ToRgb Operation.

2.2.0

1.9.0

API Updates

  • QnnGpuOpPackage.h: Support memory object offsets and layouts.

OpDef Updates

  • Support INT32 in ElementwiseAdd Op [broadcast only]

2.1.0

1.8.0

OpDef Updates

  • Added support for LayerNorm Operation.

  • Added support for UnPack Operation.

  • Added support for DetectionOutput Operation.

  • Added support for SquaredDifference Operation.

  • Added half_pixel_centers param support for ResizeNearestNeighbor Operation

API Updates

  • QnnGpuContext.h: Initial release. Added performance hint controls.

  • QnnContext.h: Support QnnContext_setConfig.

2.0.0

1.7.2

API Updates

  • QnnGpuOpPackage.h: Super node optimization.

  • QnnGpuGraph.h: Removed deprecated precision modes.

  • QnnGpuOpPackage.h: Update op pkg interface to use null-terminated arrays of pointers.

OpDef Updates

  • Support ElementWiseCos.

1.5.0

1.6.0

OpDef Updates

  • TransposeConv2d support output_padding param.

  • Reshape support INT32 input and output.

  • Fix issue when Cast operation has input dims larger than 32768.

1.4.0

1.5.0

OpDef Updates

  • Added Half Pixel Centers param support for Resize Operation.

    • This param is included only for ResizeBilinear Operation.

1.3.0

1.4.2

API Updates

  • QnnGpuOpPackage.h: Updated QnnGpu_DeviceProperties_t

    • Added maximum height, width, and depth fields for Image2dArray memory objects.

OpDef Updates

  • Support added for ElementWiseRound.

  • Relax constraint on Concat axis param to axis >= N - 4.

1.2.0

1.3.0

API Updates

  • QnnGpuGraph.h: Added disableQueueRecording field to QnnGpuGraph_CustomConfig_t

    • Provide an option to disable the recordable command queue feature.

1.1.0

1.1.0

API Updates

  • QnnGpuGraph.h: Deprecated and added precision modes

    • Deprecated QNN_GPU_PRECISION_FULL_FLOAT

    • Deprecated QNN_GPU_PRECISION_HALF_FLOAT

    • Deprecated QNN_GPU_PRECISION_DEFAULT

    • Added QNN_GPU_PRECISION_FP32 (equivalent to QNN_GPU_PRECISION_FULL_FLOAT)

    • Added QNN_GPU_PRECISION_FP16 (equivalent to QNN_GPU_PRECISION_HALF_FLOAT)

    • Added QNN_GPU_PRECISION_USER_PROVIDED (equivalent to QNN_GPU_PRECISION_DEFAULT)

  • QnnGpuOpPackage.h: Add tierName field to QnnGpu_DeviceProperties_t

    • Provides the GPU tier (e.g. 660). Requires rebuild of 3rd party op packages.

  • QnnGpuOpPackage.h: Add sdkApiVersion field to QnnGpuOpPackage_GlobalInfrastructure_t

    • Provide QNN API and QNN GPU API versions to op packages

OpDef Updates

  • Cast

    • Added QNN_DATATYPE_UINT_32 support for in[0] and out[0]

  • Dequantize

    • Bugfix for incorrect fixed point in[0] data types

  • Pad

    • Added support for EDGE scheme param

  • Transpose

    • Added support for in[0] 5-D shape

1.0.0

1.0.0

  • Initial release
