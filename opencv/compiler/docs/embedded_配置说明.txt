# 嵌入式平台 CMake 配置说明

## 配置文件: embedded_options.txt

此配置文件针对嵌入式平台（QNX、AX620A、iOS）优化，重点：
- 最小化库文件大小
- 降低内存使用
- 优化嵌入式性能
- 禁用不必要的功能

### 核心优化策略

```
# 嵌入式平台优化
-DBUILD_SHARED_LIBS=OFF            # 静态链接减小体积
```

### 图像格式支持（最小化）

```
# 仅启用必需的图像格式
-DBUILD_JPEG=ON                    # JPEG 支持（基本必需）
-DBUILD_PNG=ON                     # PNG 支持（基本必需）

# 禁用大型图像格式以节省空间
-DBUILD_TIFF=OFF                   # TIFF 支持（体积大）
-DBUILD_WEBP=OFF                   # WebP 支持（可选）
-DBUILD_OPENEXR=OFF                # OpenEXR HDR（体积大）
-DBUILD_JASPER=OFF                 # JPEG-2000（复杂）
-DBUILD_OPENJPEG=OFF               # 开源 JPEG（可选）
```

### 视频支持（禁用）

```
# 禁用视频处理（嵌入式设备通常不需要）
-DWITH_FFMPEG=OFF                  # FFmpeg 视频编解码（体积大）
-DWITH_GSTREAMER=OFF               # GStreamer 框架（复杂）
-DWITH_AVFOUNDATION=OFF           # macOS 视频框架（不适用）
-DWITH_GTK=OFF                    # GTK 图形界面（不适用）
-DWITH_V4L=OFF                    # Video4Linux（不适用）
```

### 并行处理（最小化）

```
# 最小化并行处理支持
-DWITH_TBB=OFF                     # Intel TBB（嵌入式可能不支持）
-DWITH_OPENMP=OFF                  # OpenMP（嵌入式可能不支持）
```

### 硬件加速（选择性）

```
# 禁用可能不可用的硬件加速
-DWITH_OPENCL=OFF                  # OpenCL（嵌入式 GPU 支持有限）
-DWITH_OPENCL_SVM=OFF             # OpenCL SVM（复杂）
-DWITH_IPP=OFF                    # Intel IPP（x86 专用）
-DWITH_VULKAN=OFF                 # Vulkan（嵌入式支持有限）
```

### 核心模块（仅必需）

```
# 仅启用核心必需模块
-DBUILD_opencv_core=ON             # 核心功能（必需）
-DBUILD_opencv_imgproc=ON          # 图像处理（必需）
-DBUILD_opencv_imgcodecs=ON        # 图像编解码（必需）

# 禁用图形界面（嵌入式通常不需要）
-DBUILD_opencv_videoio=OFF         # 视频 I/O（通常不需要）
-DBUILD_opencv_highgui=OFF         # 高级 GUI（不需要）
```

### 选择性额外模块

```
# 根据需求选择启用
-DBUILD_opencv_calib3d=OFF         # 相机标定和 3D 重建（可选）
-DBUILD_opencv_features2d=ON       # 2D 特征处理（常用）
-DBUILD_opencv_flann=ON            # 快速近似最近邻（轻量）
-DBUILD_opencv_ml=OFF              # 机器学习（体积大）
-DBUILD_opencv_objdetect=ON        # 目标检测（常用）
```

### 重型模块（禁用）

```
# 禁用计算密集和体积大的模块
-DBUILD_opencv_dnn=OFF             # 深度学习（体积大，计算密集）
-DBUILD_opencv_gapi=OFF            # 图形 API（复杂，体积大）
-DBUILD_opencv_stitching=OFF       # 图像拼接（计算密集）
-DBUILD_opencv_video=OFF           # 视频分析（可选）
-DBUILD_opencv_world=OFF           # 世界库（包含所有模块）
```

### 开发功能（禁用）

```
# 禁用开发相关功能以减小体积
-DBUILD_TESTS=OFF                  # 单元测试（部署不需要）
-DBUILD_PERF_TESTS=OFF            # 性能测试（部署不需要）
-DBUILD_EXAMPLES=OFF              # 示例程序（部署不需要）
-DBUILD_DOCS=OFF                  # 文档（部署不需要）
```

### 语言绑定（禁用）

```
# 禁用语言绑定（嵌入式通常使用 C++）
-DBUILD_opencv_python3=OFF        # Python 3 绑定（不需要）
-DBUILD_opencv_java=OFF           # Java 绑定（不需要）
-DBUILD_opencv_objc=OFF           # Objective-C 绑定（仅 iOS 需要）
```

### 大小优化

```
# 额外的体积优化
-DENABLE_PRECOMPILED_HEADERS=OFF  # 预编译头文件（节省空间）
-DENABLE_FAST_MATH=ON             # 快速数学运算（性能优化）
```

## 平台特定考虑

### iOS 平台
- 可能需要启用 `BUILD_opencv_objc=ON` 如果需要使用 Objective-C
- 考虑使用 Bitcode 支持（如果需要）
- 注意 App Store 的大小限制

### QNX 平台
- 确保所有依赖库在 QNX 上可用
- 可能需要自定义工具链配置
- 注意 QNX 特定的系统调用

### AX620A 平台
- 针对特定芯片优化的编译选项
- 可能需要特殊的交叉编译工具链
- 注意内存和存储限制

## 使用建议

### 最小化配置
```bash
# 使用默认的嵌入式配置
./build.sh -t <target> -r
```

### 自定义模块
```bash
# 添加特定需要的模块
./build.sh -t <target> -r -a "-DBUILD_opencv_video=ON"
```

### 调试版本
```bash
# 构建调试版本（注意会增加体积）
./build.sh -t <target> -d
```

## 性能优化提示

1. **内存使用**: 静态链接减少了运行时内存碎片
2. **启动时间**: 最小化模块数量可以加快库加载
3. **CPU 使用**: 禁用不需要的模块减少 CPU 开销
4. **存储空间**: 优化后的库可能只有完整版本的 1/5 到 1/10

## 典型应用场景

### 移动机器人
- 启用目标检测和基本图像处理
- 禁用视频和深度学习功能
- 重点优化实时性能

### 嵌入式视觉系统
- 仅启用特定的图像处理算法
- 禁用所有 GUI 相关功能
- 优化内存使用

### IoT 设备
- 最小配置以节省资源
- 仅启用最基本的功能
- 重点优化功耗

## 故障排除

### 链接错误
- 确保目标平台支持所有启用的功能
- 检查交叉编译工具链配置
- 验证所有依赖库都可用

### 运行时错误
- 检查目标平台的库依赖
- 验证硬件支持的功能
- 确保足够的内存和存储空间
