# OpenCV 构建系统 - 快速开始指南

本指南提供了为不同平台构建 OpenCV 的快速说明。

## 前置要求

### 必需工具
- CMake 3.10 或更高版本
- GNU Make 或等效工具
- Git（用于源代码管理）

### 平台特定要求

#### Android
- Android NDK（设置 `ANDROID_NDK_HOME` 环境变量）
- 最低 API 级别 21（Android 5.0）

#### iOS
- 带 iOS SDK 的 Xcode
- iOS 工具链文件

#### QNX
- QNX SDK（设置 `QNX_HOST` 和 `QNX_TARGET` 环境变量）

## 快速构建命令

### 1. 构建当前平台（macOS/Linux）
```bash
cd compiler
./build.sh
```

### 2. 构建 Android ARM64 (arm64-v8a)
```bash
cd compiler
./build.sh -t 2 -r -n opencv -v 4.11.0
```

### 3. 构建 Android ARMv7 (armeabi-v7a)
```bash
cd compiler
./build.sh -t 1 -d -n opencv -v 4.11.0
```

### 4. 构建 iOS ARM64
```bash
cd compiler
./build.sh -t 8 -r -n opencv -v 4.11.0
```

### 5. 构建到自定义安装路径
```bash
cd compiler
./build.sh -t 2 -r -i /opt/opencv/android
```

## 构建类型

- `-r, --release`: Release 构建（优化，默认）
- `-d, --debug`: Debug 构建（包含调试符号）
- `--RelWithDebInfo`: 带调试信息的 Release 版本

## 常用选项

```bash
# 显示帮助和所有选项
./build.sh --help

# 列出所有可用目标
./build.sh --list-targets

# 使用额外 CMake 参数构建
./build.sh -t 2 -r -a "-DWITH_CUDA=ON -DBUILD_opencv_dnn=ON"

# 构建带详细输出的调试版本
./build.sh -t 2 -d -n opencv -v 4.11.0
```

## 输出位置

### 构建文件（中间文件）
```
build/<platform>-<arch>-<type>/
# 示例: build/android-arm64-v8a-release/
```

### 最终库文件
```
lib/<version>/<platform>-<arch>-<type>/
# 示例: lib/v4.11.0/android-arm64-v8a-release/
```

## 环境变量

### Android
```bash
export ANDROID_NDK_HOME="/path/to/android-ndk"
```

### QNX
```bash
export QNX_HOST="/path/to/qnx/host"
export QNX_TARGET="/path/to/qnx/target"
```

## 故障排除

### 常见问题

1. **找不到 CMake**: 安装 CMake 3.10+
2. **Android NDK 未设置**: 设置 `ANDROID_NDK_HOME` 环境变量
3. **构建目录已存在**: 删除或备份现有构建目录
4. **权限被拒绝**: 确保脚本可执行 (`chmod +x build.sh`)

### 获取帮助

```bash
# 显示详细帮助
./build.sh --help

# 检查可用目标
./build.sh --list-targets

# 检查错误日志
tail -f build.log
```

## 项目集成

### CMake 集成
使用根目录中的 `opencv.cmake` 文件与您的 CMake 项目集成：

```cmake
include(/path/to/opencv/opencv.cmake)
```

### 手动集成
链接到 `lib/` 目录中的构建库：

```cmake
target_link_libraries(your_target
    ${OPENCV_LIB_DIR}/libopencv_core.so
    ${OPENCV_LIB_DIR}/libopencv_imgproc.so
    # ... 其他 OpenCV 库
)
```

## 性能提示

1. **并行构建**: 脚本自动使用最优线程数
2. **清理构建**: 删除构建目录进行完全重新构建
3. **增量构建**: 重新运行脚本将执行增量构建
4. **磁盘空间**: 构建目录可能很大（每个平台 500MB-2GB）
